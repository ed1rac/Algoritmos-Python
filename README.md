# Algoritmos-Python

Esse projeto é uma ideia de criar programas em <u>Python</u> para o ensino de Algoritmos e Estruturas de Dados.<br>
O livro usado como guia é o "Entendendo Algoritmos: um guia ilustrado para programadores e outros curiosos" de Aditya Y. Bhargava com muitas modificações dos programas.<br>
A ideia é compilar um curso de Estruturas de Dados em Python (por enquanto, na versão 2.7) (incluindo as transparencias)
